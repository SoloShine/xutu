# 泛化验证报告：《生死场》

日期：2026-05-18

## 核心结论

**三层图谱架构对不同风格/主题的小说同样有效。** 用萧红《生死场》（散文化、群像式、农村题材）验证，1255字图谱上下文准确预测了第13章的核心情节线，风格层成功阻止了抒情独白和知识分子话语。

## 验证方法

- 素材：萧红《生死场》，17章，~5.1万字（繁体中文，维基文库公版）
- 图谱范围：前12章（25事件/12人物/10地点/6风格规则/5意象/12章弧线，105关系）
- 测试：用图谱上下文续写第13章，对比原文ground truth（~5700字）

## 对比结果

| 维度 | 图谱续写版 | 原文 | 评估 |
|------|-----------|------|------|
| 核心美学(冷客观) | 无渲染，冷静叙述 | 一致 | 匹配 |
| 情节方向 | 搜查→集会→藏枪→宣誓→二里半拒绝 | 一致 | 匹配 |
| 对话风格 | 极简方言 | 一致 | 匹配 |
| 意象运用 | 核心意象在，细节意象少 | 更丰富 | 部分匹配 |
| 人物行为 | 趙三/王婆/二里半全部正确 | 一致 | 匹配 |
| 篇幅 | ~2200字 | ~5700字 | 偏短60% |

## 与《间隙》对比

| 指标 | 《间隙》(v2) | 《生死场》(v5) |
|------|-------------|---------------|
| 小说类型 | 文学/内心独白/单POV | 散文/群像/历史 |
| 风格 | 极简主义 | 冷客观+感官描写 |
| 字数 | 3.5万字 | 5.1万字 |
| 图谱上下文 | 1882字 | 1255字 |
| 压缩比 | 49:1 | 45:1 |
| 核心美学 | 通过 | 通过 |
| 情节方向 | 通过(有arc时) | 通过(无arc也通过) |
| 结尾锚点 | 精确(有arc时) | 精确 |

## 关键发现

1. **风格层是跨作品通用的最高ROI层**：6条规则成功约束了萧红风格（冷客观、感官细节、禁止抒情），与间隙的风格层作用一致
2. **情节层在没有arc时仍能工作**：生死场的测试没有为ch13构建ChapterArc，但通过主题线"抗爭"和前一章事件的引导，情节方向仍然正确
3. **篇幅是主要差距**：续写只有原文的40%，缺少场景展开的细节——这可能是prompt缺少场景级指导导致的
4. **意象层有效但不够**：核心意象（山羊、红蜡烛、血）出现，但原文中大量的过渡性意象（蝴蝶、瓜田、母猫、树叶）缺失
5. **图谱schema完全复用**：graph.py的8种节点类型无需修改即可适配完全不同的小说

## 模板适应性

《间隙》→《生死场》的模板变化：
- 移除gap相关字段（间隙概念不通用）
- 风格规则完全重写（6条新规则）
- 意象和主题完全不同
- ChapterArc格式一致，可直接复用
- 提取prompt的粒度规则和地点规则可通用
