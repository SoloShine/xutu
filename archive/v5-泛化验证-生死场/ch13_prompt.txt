【時間段】九一八後：侵略與抗爭 (1931-1934)
【主題】亡國與反抗

【前一章事件】
  - 日本人宣傳王道: 日本人用飛機撒傳單宣傳'王道'，紅色綠色的紙片播種子，村人失去心的平衡
  - 日本兵搜查村莊: 日本兵搜查村莊，帶走女人，被帶走的女人曲背如豬

【人物狀態】
  王婆(主角): 趙三妻子，堅強潑辣，經歷過多重苦難（殺嬰、服毒、喪子），教導女兒報仇。村中最具反抗意識的女人
  趙三(主角/農民): 剛烈的老農民，年輕時組織鐮刀會，失敗後變得怯懦。日本入侵後重新覺醒，組織革命軍

【核心地點】
  村莊: 哈爾濱近郊的偏僻農村，故事主要發生地

【主題線】
  生存: 農民在貧困中的頑強生存，像動物一樣活著 (1-9章)
  死亡: 死亡的普遍性——殺嬰、瘟疫、屠殺、自殺 (1-17章)
  女性苦難: 生育的刑罰、家庭暴力、被迫賣身 (1-16章)
  抗爭: 從鐮刀會到革命軍，從個人反抗到民族覺醒 (4,8,13-17章)

【敘事風格（必須嚴格遵守）】
  - 第三人稱全知視角，但聚焦於感官細節而非心理分析。不進入任何角色的內心獨白。用動作和環境暗示情感
  - 短句與長句交替。描寫時節奏舒緩如水流，事件時節奏急促。句子有重量，像石頭落入水中。不追求華麗
  - 細節來自動物、植物、天氣、身體。用具體的感官描寫代替抽象的情感詞。'汗水如珠如豆'比'她很累'好
  - 冷靜客觀地書寫殘酷。不煽情不評判不感嘆。人物死亡時不寫'悲慘'，只寫看見的事實。重量藏在輕描淡寫裡
  - 人物通過動作和對話表達情感，不通過內心獨白。對話極簡，方言口語化。罵人是最常見的交流方式
  - 禁止：寫知識分子式的抒情獨白；讓農民說出覺悟性的政治話語；用現代都市感的比喻；跳過苦難用浪漫化解；讓敘述者替人物感嘆命運

【核心意象】
  動物: 動物與人並置，人如動物般生存。山羊是二里半最後的牽掛，老馬是弱者命運的象徵
    演變: 山羊（日常財產）→ 老馬（被屠殺）→ 狗（食屍）→ 羊群（童年純真）→ 蚊蟲（騷擾不安）
  血/死亡: 死亡是日常的一部分，不分人和動物。亂墳崗子是死亡的中心意象
    演變: 殺嬰的血 → 鐮刀會的血 → 難產的血 → 服毒的掙扎 → 瘟疫的屍體 → 屠殺的血
  季節/自然: 季節循環是生死的節奏。自然不關心人的苦難，只是靜靜地變換
    演變: 夏天的菜圃 → 冬天的荒山 → 五月節 → 蚊蟲的夏 → 傳染病的秋 → 雪天的旗
  身體: 身體是苦難的載體。殘疾、疾病、生育、暴力都刻在身體上
    演變: 羅圈腿的腿 → 月英被磚塊壓的身體 → 孕婦的肚子 → 平兒多肉的肩 → 二里半不健全的腿
  火/煙: 煙是日常的，火是抗爭的。從廚房煙囪到革命誓言的蠟燭
    演變: 麻面婆燒飯的煙 → 紅蠟燭的火焰 → 革命的火種
