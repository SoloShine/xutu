**verdict**: **异质明显优 + scale_signal = "放大"** ⭐⭐⭐

## §0. 一句话结论

**中场景 verdict = 异质明显优，scale_signal = 放大。** 这是本研究最强的正面结果——在 3 层集体嵌套 + 8 角色 + 10 ticks 的中规模场景下，异质系统产生了 10 项独有的层级冲突 + 9 组嵌套集体冲突 + 9-tick 集体演化深度（超越小场景基准 4）。**更关键的是**：即使修正了同质 prompt（赋能式，给同质 agent 充分自主判断空间，不预先剥夺跨层能力），同质虽然涌现了集体行为（L2×32 trace），但仍输——**这证明异质优势来自架构本身（独立层级实体间的不可还原冲突），而非 prompt 引导**。

**用户洞察被验证**：小场景（5×4）→ 中场景（13×10）复杂度放大，异质优势也随之放大（不是崩溃，不是被追上）。thesis 1 从"条件性成立"升级为"规模放大 + prompt 公平双验证成立"。

## §4. 嵌套集体冲突（异质 9 组 vs 同质 8 组）

### 4.1 异质独有的嵌套集体冲突（核心差异化）

```
小队(L2_base)保护韩峥 vs 网络(L2_mid)立约绝对禁止任何牺牲 + 小队认为网络"不够决断"
小队(L2_base)单点保护 vs 网络(L2_mid)保全体成员——视角嵌套张力(救陆 vs 救韩的零和取舍)
联盟(L2_top)接受韩峥个人决断被网络视为对牺牲的容忍 vs 网络立约级禁止
联盟(L2_top)容器化框架部署 vs 网络(L2_mid)立约反对牺牲——联盟"不阻止个人自愿"被网络视为二次容忍
联盟(L2_top)"控制=接纳方向隐性预设"自我对质 vs 联盟自身核心规则"维持文明存续"(顶层集体内部自我消解张力)

网络立约修正案"穷尽前不执行" vs 小队提请会议时限——嵌套集体的时间尺度张力
小队核心规则"韩峥不死"条件性分化预案 vs 命运层裁决共担——基层集体核心规则面临结构性失效前的根本内部张力
```


### 4.3 核心差异（公平对比的关键）

同质配置确实涌现了 8 组嵌套集体冲突（数量接近异质的 9 组）——**这是赋能式 prompt 修正的价值**：移除剥夺式抑制后，同质 agent 能自主形成集体行为。

**但性质不同**：
- **同质的"集体"** = 个体 agent 通过程序咬合形成的事实集合（议长+代表+协调者+成员的协商）
- **异质的"集体"** = 独立集体实体的立场嵌套冲突（立约修正案、规则自我消解相、核心规则存续边界等**集体内部规则演化**）

同质的联盟代表只能"个人声明"，无法让联盟作为实体"自我消解法理人格"。异质的联盟集体可以主动请求命运层裁定自身合法性——这是集体作为实体的行为，不是个体代表的协商。

## §5. 赋能式同质的涌现（公平性验证）⭐

### 5.1 赋能式 prompt 修正的价值

补强发现原同质 prompt 是剥夺式（"你没有特殊地位/世界必然性对你只是传闻"），预先封死涌现路径。中场景使用了**赋能式修正**：

> "你的判断完全自主。你可以：只关注个人目标（layer 填 L1）/ 或尝试察觉集体共识，代表集体发声（layer 填 L2）/ 或尝试把握世界必然性（layer 填 L3）/ 或尝试校验规则（layer 填 cross_cut）。没有人预先否定你任何能力。"

### 5.2 修正后的同质涌现

赋能式同质确实涌现了非 L1 的 trace：
- **L2×32**（vs 异质 L2×28——甚至超过异质！）
- **cross_cut×42**（vs 异质 cross_cut×20——远超异质！涌现了集体观察者 A/B、规则思考者、世界观察者等横切角色）
- **L3_explicit×2**（vs 异质 L3×10——仍有差距）

**这是公平对比的关键证据**：同质在赋能式 prompt 下并未被剥夺能力，它确实自主尝试了跨层行为。

### 5.3 但同质仍输的原因

同质涌现的 L2/cross_cut trace，本质是**个体 agent 自标层级**（"我觉得我的行动属于 L2"），而非**独立层级实体的行为**。

- 同质的"联盟代表个人"声明 L3，不等于联盟作为集体实体的 L3 行为
- 同质的"规则思考者"是 agent 自封的横切角色，不等于独立的 LawEnforcer 实体
- 同质无独立 WorldWill 实体——它只有 2 条 agent 自标 L3，无法产生"命运层物理排斥非自愿路径"这类裁决

**结论**：赋能式 prompt 修正排除了"prompt 偏差作为异质优势主因"。异质优势来自**架构本身**——独立层级实体间的不可还原冲突。

| 配置 | 集体演化深度 | 演化路径 |
|------|------------|---------|

## §7. scale_signal 判定：放大 ⭐⭐⭐

用户洞察"小场景 vs 大场景复杂度差异会很大"的三种可能结果：

| 可能 | 中场景实际 | 判定 |
|------|----------|------|
| A. 放大（复杂度上升→异质优势更强）| ✅ 异质独有冲突 10 项 / 嵌套集体冲突 9 组 / 演化深度 9 tick（均超小场景）| **成立** |
| B. 崩溃（agent 太多→协调成本超过收益）| ❌ trace 连贯，无失焦，10 tick 集体立场清晰演化 | 排除 |
| C. 追上（同质在大场景涌现集体行为）| ⚠️ 同质确实涌现集体行为（L2×32），但仍是"个体咬合"非"实体冲突" | 部分追上但本质差距仍在 |

**判定 = 放大**。规模放大了异质优势——多层集体嵌套在 10 ticks 下产生了小场景无法容纳的演化深度。

## §8. 对 thesis 1 的判定（升级）

### 8.1 thesis 1 从"条件性成立"升级为"规模放大 + prompt 公平双验证成立"

**升级证据**：
1. **规模维度**：小场景（5×4，异质明显优）→ 中场景（13×10，放大）—— 异质优势随规模放大，不崩溃
2. **公平性维度**：剥夺式同质 prompt（原 pilot）→ 赋能式同质 prompt（中场景）—— 移除 prompt 偏差后异质仍优
3. **架构本质**：同质即使赋能涌现集体行为，仍是"个体咬合"非"实体冲突"——异质优势来自独立层级实体

### 8.2 最强涌现（中场景独有）

- **联盟"核心规则自我消解相"**：集体主动请求命运层裁定自身合法性边界——这是集体作为实体的最深行为
- **命运层裁决相→汇聚相→终局清算相→物理落地**：WorldWill 从被动诊断升级为主动裁定（改变世界状态）
- **"必然性以非事件形式兑现"**：集体程序严格性迫使必然性从"事件性单点死亡"改写为"分布式承担"——这是哲学级的涌现
- **立约从绝对禁止→修正案→释义权移交→见证者锁定**：集体规则的实体化演化

### 8.3 诚实标注（局限）

1. **单场景**：中场景只测了 1 个（封印危机扩展版）。需扩展到其他场景类型。
2. **大场景未验证**：~30+ agent × 20+ ticks 的大场景仍未跑。中场景"放大"能否外推到大场景？开放问题。
3. **成本超预期**：中场景 16.4M token（预估 5M）——trace 内容极丰富导致。大场景可能 40M+ token，需评估是否值得。
4. **同质"个体咬合"质量**：同质涌现的 L2×32 trace 质量如何？部分可能是 agent 自标层级，不一定是真集体行为。需更深审计。
